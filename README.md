### PDK_ROOT

`PDK_ROOT` must point to the root of your SKY130A PDK installation before running any simulation target.

```bash
export PDK_ROOT=/path/to/sky130A
# Example (volare install): export PDK_ROOT=$HOME/.volare/sky130A
# Example (open_pdks):      export PDK_ROOT=/usr/local/share/pdk/sky130A
```
